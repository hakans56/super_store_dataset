
# Superstore Sales Analysis (Excel & Tableau)

This project analyzes retail sales data from a Sample Superstore dataset (Kaggle).

## Tools
- Excel (data cleaning & exploratory analysis)
- Tableau (data visualization & dashboard)
- GitHub (version control & portfolio)

## Key Questions
- Which regions generate the highest sales and profit?
- Which categories and sub-categories are the most/least profitable?
- Are there products or categories that consistently generate losses?

## Files
- SampleSuperstore : Raw data downloaded from Kaggle
- SampleSuperstore_cleaned: Excel file with pivots and basic analysis
